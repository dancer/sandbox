# Examples

Minimal starting points for each adapter.

Run them from an app that has the relevant adapter package installed and the
provider credentials configured. The local example runs without remote
credentials.

- `core.ts`
- `blaxel.ts`
- `codesandbox.ts`
- `e2b.ts`
- `daytona.ts`
- `vercel.ts`
